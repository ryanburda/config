# config

Terminal configuration and dotfiles.

Local installation on macOS: (intended to be cloned into `~/Developer/config`
```
mkdir ~/Developer
cd ~/Developer
git clone https://github.com/ryanburda/config.git
cd config
./setup.sh
```

Optional repo setup:
```
./setup_repos.sh
```

Containerized environment: (Work in progress)
```
make build
make run
```
